# 技能帮小程序后端接口

## 响应

### 当请求成功时，返回以下内容

+ Response 200

        {
            "success": 1,
            "data":
            {
                // 数据
            }
        }

### 当请求失败时，返回以下内容

+ Response 200

        {
            "success": 0,
            "err_msg": "报错信息"
        }
